Personal Expense Tracker

This is a simple Flutter app I made to help track daily income and expenses.

📌 What it does
Add income and expense transactions
Edit or delete entries
Shows simple monthly summary (income, expense, balance)
Basic categories like food, travel, bills, etc.
🛠️ Tech used
Flutter
Dart
🎯 Purpose

I built this project to practice Flutter basics and understand how to manage data in a simple app.
